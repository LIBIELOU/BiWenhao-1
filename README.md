# 静态网站导出

这是毕文浩个人作品展示网站的静态版本。

## 使用方法

1. 直接将网站文件上传到任何静态网站托管服务
2. 访问 `index.html` 即可查看网站

## 文件说明

- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - JavaScript 逻辑
- `data.js` - 静态数据文件（包含所有内容数据）
- `modules/` - 功能模块目录

## 特性

- ✅ 完全静态，无需后端服务器
- ✅ 所有数据已嵌入，可直接使用
- ✅ 支持视频播放、图片展示等功能
- ✅ 响应式设计，支持移动端

## 更新数据

如需更新网站内容，请：

1. 修改 `config/website-content.json` 文件
2. 运行 `node export-static.js` 重新导出
3. 上传新的 `dist` 目录内容
