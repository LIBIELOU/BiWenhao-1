🚀 部署方法
=====================================

方法 1：GitHub Pages（推荐）
--------------------------------------
1. 将本文件夹内容上传到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 选择 "Deploy from a branch"
4. 选择 main 分支和 / (root) 目录
5. 访问：https://your-username.github.io/your-repo/

方法 2：直接打开
--------------------------------------
双击 index.html 文件即可在浏览器中打开

方法 3：本地服务器
--------------------------------------
1. 安装 Python
2. 在文件夹中运行：python -m http.server 8080
3. 访问：http://localhost:8080
